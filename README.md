## 开发

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
```

默认本地开发地址：

- <http://localhost:5173>

## 内容位置

- `src/data/site/config.ts`：站点基础配置
- `src/data/site/about.md`：About 页面内容
- `src/data/site/*.ts`：友链、项目、基础设施等静态数据
- `src/data/docs/posts/*.md`：文章
- `src/data/docs/notes/*.md`：笔记

模板相关说明请看 `diesuwa-starter`：
